# day 9 - linux 命令 : `cat`

`cat` 命令用来显示文件内容

## 语法

```
cat [选项] 文件……
```

## 选项
    
+   `-n, --number`

    从 1 开始对所有输出的行数进行编号
    
+   `-b, --number-nonblank`

    从 1 开始对所有输出的行数进行编号，但不对空白行编号
    
+   `-s, --squeeze-blank`

    当遇到两行以上的空白行，就替换为一行的空白行
        
## 实例

+   查看文件 `a` 的内容

    `cat a`
    
+   同时显示文件 `a` 和文件 `b` 的内容
    
    `cat a b`
    
+   将文件 `a` 和 `b` 合并后放入 `c` 中

    `cat a b > c`
